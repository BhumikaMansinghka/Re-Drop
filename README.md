ReDrop is a smart RO (Reverse Osmosis) monitoring system designed to improve water purification efficiency using an eco-friendly bio-carbon pre-filter and IoT-based real-time monitoring.

It helps users track water quality, optimize filter usage, and reduce maintenance costs through smart alerts and predictive insights.

🚀 Features
🌱 Eco-friendly bio-carbon pre-filter
📊 Real-time water quality monitoring
📡 IoT-enabled live data tracking via Wi-Fi
🔔 Filter health alerts & replacement notifications
⚙️ Predictive maintenance system
📱 Dashboard for data visualization
